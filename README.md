# Shiny Dhar Portfolio

A React + Vite portfolio site for showcasing work in AI governance, data strategy, policy, and responsible technology.

## Tech stack

- React
- Vite

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Project structure

```text
Portfolio Website/
  public/
  src/
    App.jsx
    index.css
    main.jsx
  ShinyDhar_Portfolio.jsx
  .gitignore
  index.html
  package.json
  README.md
  vite.config.js
```

## Deployment

This project is ready to deploy on Vercel as a Vite app.
